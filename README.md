Spotify Gesture Controller

A Computer Vision based Spotify controller built using Python, OpenCV, MediaPipe and Keyboard

Features

- Palm : Play/Pause
- Thumb Right : Next Track
- Thumb Left : Previous Track
- One Finger : Volume Down
- Two Fingers : Volume Up

Technologies Used

- Python
- OpenCV
- MediaPipe
- Keyboard
